# webpage

Personal website at [veiback.se](https://veiback.se), built with Astro. Bilingual: `/sv/` and `/en/`.

## Deploy

```sh
dev build
bash scripts/deploy.sh
```

## Data

| File | Description |
|------|-------------|
| `src/data/content.sv.yaml` | Site content in Swedish |
| `src/data/content.en.yaml` | Site content in English |
| `resume.yaml` (repo root) | CV data shared with `cv/` — edit only there |
